# watir-test

Watir POC with gmail

## Requirements

* ruby 2

## Quick start

Clone the project:

```
git clone https://github.com/alannadenise/watir-test.git && cd watir-test
```

Get bundler:

```
sudo gem install bundler
```

Execute gemfile:

```
sudo bundle install
```

Run script:

```
ruby test.rb
```

OR

```
ruby testproxy.rb
```
